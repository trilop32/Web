Дескрипторы (Tags)

<p>(Paragraph)(Obsolte) - блочный дескриптор который описывает абзац, и поддерживает aтребуты:
align= "left |right| center | justife(по ширене)" - выравгнивние абзаца
<b>(Bold) - линейный дескриптор, который делает текст курсивным
<i>(Italic) - линейный дескриптер, который делает текст курсивом
<r>(Underscore) - линейный дискриптор, который подчёркивает текст
<s>(Strike) - линейный дискриптор,который перечёруивает текст
<strike>(Strike)(Obsolte) - линейный дискриптор, который перечёркивает текс

Числовые типы

Целочисленные типы

  • short (2 Байта памяти)
    • unsigned short: 0…65 535; 0000 0000 0000 0000 .......... 1111 1111 1111 1111 0…216–1;
    • signed short: –32 768 … 32 767; –215 … +215 –1;
  • long (4 Байта памяти)
    • unsigned long: 0 … 4 294 967 295; 0 … 232-1;
    • signed long: – 2 147 483 648 … 2 147 483 647; – 231 … 231–1;
  • int (Integer – Целое число) (Платформенно зависимый тип)

    В Visual Studio для Microsoft Windows тип данных int занимает 4 Байта, следовательно, его диапазоны принимаемых значений полностью совпадают с long.

  • long long (8 Байт памяти)

    • signed long long int: –263 … 263 – 1;
